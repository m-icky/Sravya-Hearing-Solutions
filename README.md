# Sravya Hearing Solutions

Sravya Hearing Solutions is a premium, fully responsive, and modern landing page for a professional hearing care and hearing aid clinic based in Thrissur, Kerala. The website is built with a focus on rich aesthetics, high performance, interactive micro-animations, and solid SEO fundamentals.

---

## 🌟 Key Features

- **Modern & Interactive Aesthetics**: Curated color palette featuring Deep Navy, Royal Blue, Vibrant Cyan, and Accent Orange with smooth gradient overlays and glassmorphism.
- **Scroll-Driven Animations (GSAP)**: Integrated GSAP (GreenSock Animation Platform) and ScrollTrigger for premium page transitions, stagger effects, and smooth element revealing.
- **Custom Interactive Elements**:
  - Interactive custom cursor with hover and click feedback.
  - Page loader with linear progress indication and logo pulsing.
  - Transparent-to-blur scrolling navigation bar.
  - Fully responsive, animated full-screen mobile menu.
  - Canvas particles background on the hero section.
  - Interactive floating cards, active soundwave animation, and magnetic hover buttons.
  - Interactive counter stats that count up on scroll.
  - Responsive process timeline with scroll progress tracker.
  - Smooth testimonial slider with chevron controls and dots pagination.
- **Direct Communication Tools**:
  - Floating WhatsApp action button (FAB) with predefined message triggers.
  - Click-to-call and mailto actions using the clinic's official coordinates.
- **Google Maps Integration**: Embedded responsive location map targeting the Thrissur clinic.
- **SEO & Performance Best Practices**:
  - Structured Metadata (Open Graph & Twitter Cards).
  - Schema Markup (`MedicalClinic` and `LocalBusiness` types) for rich search snippet indexing.
  - Semantic HTML5 layout structure.

---

## 🛠️ Technology Stack

- **Markup**: HTML5 (Semantic elements)
- **Styles**: Custom Vanilla CSS3 (Custom Variables, CSS Grid, Flexbox, Transitions)
- **Interactions**: Vanilla JavaScript (ES6)
- **Libraries**:
  - [GSAP (GreenSock)](https://greensock.com/gsap/) (Core, ScrollTrigger, TextPlugin)
  - [Font Awesome v6.5.1](https://fontawesome.com/) (Vector Icons)
  - Google Fonts (DM Sans)

## 🚀 How to Run Locally

Since this is a lightweight static website, it does not require any build tools or compilation steps. 

### Method 1: Python HTTP Server (Recommended)
Navigate to the project directory and run:
```bash
python3 -m http.server 8000
```
Then open [http://localhost:8000](http://localhost:8000) in your web browser.

### Method 2: Direct Open
Simply double-click the `index.html` file to open it directly in any modern web browser. (Note: Some features like canvas particles or local resources might require a local server environment due to browser CORS policies).
